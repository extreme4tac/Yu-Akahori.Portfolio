DESCRIPTION
PR

私の強みは人の話を聞く力です。 ウェブサイト制作のプロジェクトを立ち上げる際、何十人もの人から想いや夢を聞き、仲間を集めました。 集まったチーム内の意思疎通を図るために、メンバー間の認識統一に尽力。当初はウェブ制作に関しての知識がなく苦戦しましたが、筋道立て、段階的に判断する考え方を身に付けることで、徐々にギャップが解消され、メンバーから本音を聞けるようになりました。本音を聞いたことで、チームとしての問題点、方向性が明確になり、円滑なコミュニケーションを取ることが可能になり、雰囲気も明るくなりました。 Web制作と関わったことでプログラミングに興味を持ち、マークアップ言語とJavascript, PHPを独学で学んでいます。 今後、IT業界では相手の本音を引き出しながら、積極的にコミュニケーションを図って、幅広い分野で経験を積みたいと考えております。
